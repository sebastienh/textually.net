
# User Interface

The interface is divided into four main sections, the [files outline](#filesOutline), the _Text_, the _Styles_ and the [_Sidebar_](#sidebar) sections. The _Text_ section contains the [Editors Panel](#editorsPanel), which is where all the editing occurs. The  _Styles_ section contains the [_Styles List_](#stylesList) and the [_Style Editor_](#styleEditor). The [_Sidebar_](#sidebar)  contains the [_Tools_](#tools) and the [_Style Picker_](#stylePicker) tabs.
